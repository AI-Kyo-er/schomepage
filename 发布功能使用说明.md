# Content Publishing System - User Guide

## Overview

The content publishing system allows users to publish their HTML articles to the main homepage (http://localhost:5000/home) with three different card types: Academic Articles, Campus Blogs, and News.

## Features

### 1. Publish Content Interface
- Location: Main editor page -> Left sidebar -> "ğŸ“¤ Publish Content"
- Users can publish their articles to the homepage
- Each publication includes:
  - Article selection
  - Content type (Article/Blog/News)
  - Title
  - Description (20-200 characters)
  - Author name
  - Cover image

### 2. Content Types

### 3. Dynamic Homepage
- URL: http://localhost:5000/home
- Automatically loads all published content from publications.csv
- Filter by content type using navigation tabs
- Beautiful card layout with hover effects
- Click cards to view full articles

## How to Use

### For Users

1. **Create an article** in the editor
2. **Click "ğŸ“¤ Publish Content"** in the left sidebar
3. **Fill in publication details:**
   - Select your article
   - Choose content type
   - Enter title (will be shown on homepage)
   - Write description (50-150 words recommended)
   - Set author name
   - Choose cover image
4. **Click "ğŸ“¤ Publish to Homepage"**
5. **Visit http://localhost:5000/home** to see your published content!

### Manage Publications

- View all your published content in the "My Published Content" section
- Click "Cancel Publication" to remove content from homepage
- Note: Canceling publication does NOT delete the article file

### For Admin

- Published content is stored in `publications.csv`
- Each publication has a unique ID
- Multiple users can publish to the same homepage
- Content is automatically filtered by user permissions

### API Endpoints

- `POST /api/publish_content` - Publish new content
- `POST /api/get_published_content` - Get user's published content
- `POST /api/unpublish_content` - Cancel publication
- `GET /api/get_all_publications` - Get all published content for homepage

## Benefits

1. **Centralized Homepage**: All published content appears on one beautiful homepage
2. **Easy Management**: Simple interface to publish and unpublish content

## Notes

- Each article can only be published once
- Descriptions should be informative but concise (20-200 characters)
- Choose appropriate cover images to make your content attractive
- Content type cannot be changed after publishing (must unpublish and republish)
- The "Visitor Channel" feature remains available for creating direct access links
